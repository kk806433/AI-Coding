# AI Programming Engineering Prompt Library 🚀
High-quality AI prompts for every stage of software development — from requirement analysis to code generation, debugging, and optimization.

---

## 📌 What is this?
This repository collects and organizes battle-tested AI prompts for programming engineering workflows. Whether you're using ChatGPT, Claude, or any other LLM, these prompts help you get accurate, production-ready code and technical solutions faster.

---

## 📂 Prompt Categories
- 📋 **Requirement Analysis Prompts**: Clarify user stories, define technical specifications, and break down complex requirements.
- 💻 **Code Generation Prompts**: Generate clean, maintainable code in Python, Java, JavaScript, and more.
- 🐛 **Debugging & Troubleshooting Prompts**: Identify bugs, analyze error logs, and optimize performance.
- 📝 **Documentation Prompts**: Generate API docs, READMEs, and technical comments automatically.
